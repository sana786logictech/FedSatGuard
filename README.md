# FedSatGuard: 

[![JISA](https://img.shields.io/badge/JISA-Elsevier-orange)](https://www.sciencedirect.com/journal/journal-of-information-security-and-applications)

## Overview
FedSatGuard is a secure federated learning framework designed for satellite-terrestrial networks with cross-layer anomaly detection, trust modeling, and adaptive aggregation under adversarial conditions.
